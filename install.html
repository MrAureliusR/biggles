1.6.6

By hand

  1. Install plotutils. Latest version is 2.5 as of 01 Sep 2007.
    • Make sure to add the directory containing the libplot shared library to LD_LIBRARY_PATH.
  2. Install numpy.
    • Make sure the numpy module is in your PYTHONPATH.
  3. Install biggles:
    4. $ tar xzf python2-biggles-1.6.6.tar.gz
$ cd python2-biggles-1.6.6/
$ python setup.py build
$ python setup.py install --prefix=/path/to/install
